This directory contains the sources of the SLP
(Straight Line Programs) application, which is designed to demonstrate
the basic concepts in lexical analysis, parsing, and AST construction.

* slp.bat is a Windows script for running the SLP application.
* slp is a C shell script for running SLP.  It has not been tested.
* slp.bash is a bash shell script for running SLP under the Cygwin
  environment.  It has not been tested.
